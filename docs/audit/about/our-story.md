#### Product Suite

The TechD CogSuite decision support tools extend Cognos Analytics features for management and administrator productivity and cost avoidance. CogSuite provides features and functions not available with Cognos Analytics through the TechD License Manager, Content Manager and Administrator software such as; licensing, content management, administration, user profiles, security levels, and configuration.

##### TechD IBM CogSuite of Tools – Decision Support for Cognos Analytics

TechD Administrator enables Cognos Analytics administrators, report authors and power users to perform everyday tasks that are traditionally tedious and time consuming in managing Cognos Analytics server environments. The TechD Administrator includes the following administrative functions:

- Easily list Objects Type, Location, package and change date
- Show Running, Scheduled, and Completed Jobs
- Generate a Framework Manager Model from a published package. Perfect if you’ve lost or are unsure which is your production version.
- Scan and list reports for specific values based on a specific package. Allows for peace of mind when changing or deleting package values.
- Generate a complete list of named users by group or role. Make sure users have the access you think they have.
- All results export easily to Excel.

TechD Content Manager gives you unprecedented ability to explore and manage Cognos Analytics content. Provides the ability to easily move Cognos Analytics objects inside a Cognos Analytics server or between Cognos Analytics servers as easily as using Windows Explorer. The TechD Content manager enables:

- Easy copy objects by dragging and dropping them or using windows copy and paste.
- Support for object migration between different versions of Cognos Analytics .
- Simple and easy user interface to explore and manage users My Folders.
- Access to orphaned Cognos Analytics users that have left the company

TechD License Manager quickly and accurately scans your Cognos Analytics environment to determine the licenses that are currently being utilized.

- Allows a complete self-audit to ensure internal compliance with IBM licensing.
- Offered as a standalone application so the results are internal only.
- Shows detailed security access by individual named user
- Available FREE of charge – There are no fees for the TechD License Manager
